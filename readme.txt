实验器材:
	探索者STM32F407开发板
	
实验目的:
	学习使用FreeRTOS空闲任务钩子函数的使用方法。
	
硬件资源:
	1,DS0(连接在PF9)，DS1(连接在PF10上)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面) 
	3,蜂鸣器(PF8)

实验现象:
	通过串口调试助手给开发板发送字符串，开发板接收到字符串根据字符串的内容
	控制开发板上的外设。没有任务运行的时候CPU会进入低功耗模式，此功能是由
	FreeRTOS的空闲任务钩子函数完成的。

注意事项:
	无.
